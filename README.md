# To-Do API with FastAPI 📝🚀

This is a simple To-Do List API built using *FastAPI* and *Python* as part of the API Fellowship Program.

---

## ✅ Features

- Add new tasks
- View all tasks
- Update task status (pending or completed)
- Delete tasks
